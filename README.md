# 🎓 Ostroh Academy – Modern Frontend Project

A modern frontend web application created as a **conceptual redesign** of the Ostroh Academy National University website.

The project focuses on **clean UI/UX**, **responsiveness**, and **multi-language support**, demonstrating modern frontend development practices.

---

## ⚠️ Disclaimer

* This is a **frontend-only project (no backend)**
* Some links, downloads, and deep pages from the original website are **not implemented**
* Some data is **approximated for demonstration purposes**

All university-related content belongs to the **National University of Ostroh Academy**.

---

## 🚀 Tech Stack

* **React 19**
* **TypeScript**
* **Tailwind CSS**
* **Framer Motion**
* **React Router 7**
* **Lucide React**
* **React Slick**

---

## 🧩 Features

* 📱 Fully responsive (desktop, tablet, mobile)
* 🌐 Language switch (Ukrainian / English)
* 🔗 Smooth scrolling and anchor navigation
* 🎨 Modern UI (glassmorphism, shadows, branding)
* ⚡ Fast and lightweight architecture

---

## 📂 Project Structure

### Core Pages

* `Home.tsx` – Landing page with news and highlights
* `Institutes.tsx` – Institutes and departments overview
* `Admissions.tsx` – Information for applicants
* `Science.tsx` – Research activities
* `UniversityHistory.tsx` – Historical timeline

### Additional Pages

* `Rectorate.tsx`, `Management.tsx` – Leadership
* `AcademicCouncil.tsx` – Academic council
* `Documents.tsx`, `PublicInfo.tsx` – Documentation
* `Museum.tsx` – Cultural section
* `Contacts.tsx` – Contact info and map

### Technical

* `LanguageContext.tsx` – Custom i18n system
* `NotFound.tsx` – 404 page
* `App.tsx`, `routes.ts` – Routing

## ⚙️ Getting Started

### 1. Install Node.js

Download the LTS version: https://nodejs.org

---

### 2. Clone the repository

```bash
git clone <your-repo-url>
cd your-project-folder
```

---

### 3. Install dependencies

```bash
npm install
```

---

### 4. Run the project

```bash
npm run dev
```

---

### 5. Open in browser

```
http://localhost:5173
```
